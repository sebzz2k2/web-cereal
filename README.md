# WEB CEREAL

### INTRODUCTION
Web Cereal get your cereal in web fast and light weight.

### SETUP ( DEV )
1. Move into cereal folder.
```
cd cereal
```
2. Install NPM packages.
```
npm install
```
3. Install tailwind.
```
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
```
4. Run your server.
```
npm run dev
```
